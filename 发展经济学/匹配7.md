### 1. 发展经济学的研究对象
*   **匹配问题：** 简述发展经济学的研究对象 / 为什么说发展经济学是较晚出现的经济学分支？
*   **根据课件整理的答案：**
    *   **研究对象：** 研究经济落后的发展中国家如何实现工业化与现代化。其核心难题在于资金不足，因此研究如何加速“资本形成”是实现经济起飞的关键前提。
    *   **理论演变：** 早期发展经济学（如20世纪50年代）过分强调资本形成是唯一的决定因素；到了70-80年代，理论逐渐演进，开始认识到技术进步、人力资本等其他内生因素的重要性，从而使学科体系更加完善。

### 2. 贫困的恶性循环理论
*   **匹配问题：** 名词解释： "贫困的恶性循环" / 简答题： "如何打破贫困的恶性循环？"
*   **根据课件整理的答案：**
    *   **定义（纳克斯，1953）：**
        *   **供给恶性循环：** 低收入 → 低储蓄能力 → 低资本形成 → 低生产率 → 低产出 → 低收入。
        *   **需求恶性循环：** 低收入 → 低购买力 → 投资引诱不足 → 低资本形成 → 低生产率 → 低产出 → 低收入。
    *   **如何打破：** 必须通过增加储蓄和投资，加速资本形成，提高生产率，从而从供给和需求两端同时切断这个循环。

### 4. 人力资本理论
*   **匹配问题：** 简答题： "为什么说人力资本是新增长理论的核心？" / 对比分析： "对比三阶段增长理论的差异"
*   **根据课件整理的答案：**
    *   **理论反思：** 课件指出，单纯增加资本而缺乏教育、文化等先决条件配合，会导致资本产出效率低下。
    *   **演进逻辑：**
        1.  **50年代后期：** 索罗“新古典增长模型”。
        2.  **70年代：** 贝克尔提出“人力资本”理论，强调人的素质。
        3.  **80年代：** 罗默、卢卡斯提出“内生增长模型”，将人力资本作为增长的核心动能。
    *   **结论：** 资本使用效率的提高不仅依赖物质资本，更取决于技术进步和人力资本。

### 5. 大推进理论
*   **匹配问题：** 名词解释： "大推进理论" / 简答题： "如何评价大推进理论？"
*   **根据课件整理的答案：**
    *   **核心观点：** 发展中国家实现工业化的首要障碍是资本形成不足。为了克服这种障碍，必须在各个工业部门**同时全面进行大规模投资**。
    *   **理论逻辑：**
        1.  **规模效应：** 投资必须达到足够大的规模，才能产生带动工业化的动力。
        2.  **互补性：** 各部门同时投资可以产生相互需求，提供市场。
        3.  **外部经济：** 这种大规模投资能克服“不可分性”，产生“外部经济”效应和“规模经济”效应。
    *   **评价：** 强调了资本形成的规模效应，但也需注意课件中提到的“反思”，即如果没有制度、教育等条件的配合，大规模资本投入的产出效率可能并不理想。

### 8. 新增长理论
*   **匹配问题：** 名词解释： "新增长理论"或"内生增长理论" / 简答题： "新增长理论与传统增长理论的区别是什么？"
*   **根据课件整理的答案：**
    *   **定义：** 以罗默、卢卡斯为代表的内生增长模型（80年代），强调资本形成的质量而非单纯的数量。
    *   **区别点：**
        *   **传统理论：** 早期发展经济学认为资本形成是经济增长的唯一或主要动能。
        *   **新增长理论：** 认为技术进步和人力资本是增长的内生动能。强调发挥企业和金融市场在资金配置上的主导作用，通过提高资本使用效率（而非单纯增加资本量）来增加产出。

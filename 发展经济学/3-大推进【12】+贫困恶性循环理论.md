# 原文
# 一、罗森斯坦-罗丹的大推进理论【原来是完整一段话没有分点】
1. 背景：罗森斯坦-罗丹在20世纪40年代中期就对东欧和东南欧战后重建问题发表了自己的观点，提出了大推进理论，
2. 理论认为，由于**资本、储蓄和市场**需求具有不可分性，**小规模、个别部门或者个别产业的投资**难以启动一个停滞的经济，
3. 要从根本上推动经济起飞，只有对所有的产业部门**同时**进行大规模投资，尤其是**基础设施部门的投资**，即采用“大推进”式的发展战略，冲破市场狭小的束缚，使各部门同时发展起来，才能摆脱停滞，进入工业化。

# 二、资本形成与纳克斯贫困恶性循环理
### **（一）资本形成促进经济发展的理论**
许多发展经济学家从理论上分析了资本稀缺对经济发展的阻碍作用，解释了发展中国家贫困的根源，论证了资本形成是经济发展决定性因素的观点。
### **纳克斯贫困恶性循环理论**
结论：低收入既是发展中国家贫穷落后的结果，也是其原因。
1. 从供给方面看，低收入导致低储蓄供给从而使资本缺乏，资本缺乏又导致低劳动生产率，进而导致低收入；
2. 从需求方面看，低收入降低了购买力，低购买力导致投资引诱不足，投资引诱不足又导致资本积累不足，进而导致生产规模减小、劳动生产率难以提高，而低生产率又导致低产出和低收入水平。
3. 如此往复，形成恶性循环。正如概括的那样：“一个国家穷是因为它穷。”
4. 而**解决问题的关键**在于**增加储蓄和投资**，而且要对许多行业同时进行大规模投资，使行业之间的产品形成相互需求，从而扩大市场容量，冲破贫困的恶性循环，促进经济发展。


# 解释
1.  **理论提出与基础：**
    该理论由埃及经济学家阿明（Samir Amin）在20世纪70年代，基于“中心—外围理论”和“不平等交换理论”进一步论述提出。
2.  **核心定义：**
    建立在不平等交换基础上的国际贸易，使得发展中国家的经济依附于发达国家。这种依附不仅限于贸易，还包括金融、技术等多方面的依附。
3.  **理论依据（原因）：**
    虽然国际间存在生产率差异，但在许多部门中，发展中国家与发达国家的“工资率差异”远大于“劳动生产率的差异”。
4.  **发展中国家的角色与后果：**
    *   **角色：** 发展中国家在垄断资本主义的全球积累中，以廉价劳动力为基础发挥双重功能：一是提供廉价的出口商品；二是保证发达国家的资本输出获得高利润率。
    *   **后果（二元化）：** 发展中国家现代部门生产率的提高伴随着非现代部门的贫困化，导致经济呈现高度的“二元化”。这意味着发展中国家非现代部门的贫困大众承担了全球不平衡发展的社会代价。
5.  **解决主张：**
    为了求得发展，该理论主张发展中国家应该与世界资本主义体系“脱钩”，斩断与帝国主义的联系。
# 原文 专栏10-3 不平等交换与依附论  
1969年，希腊经济学家埃曼纽尔(Emmanuel)提出了不平等交换理论。
这一理论以劳动价值论为基础，认为发达国家和发展中国家之间的贸易交换是不平等的。
由于资本可以在国际间自由流动，中心和外围国家的利润率将趋于均等。
但劳动力不能在国际间自由流动，因而工资水平的国际差异是持久的。
发达国家的出口商品含有较高的工资成本，而发展中国家的出口商品含有较低的工资成本。这就使得标价相同的商品中，发展中国家的出口商品含有更多的价值(劳动)。发达国家与发展中国家按照市场经济的等价交换原则进行贸易，但实际上交换的价值是不等的，发展中国家的剩余价值被发达国家榨取。
这就解释了外围国家经济停滞不前的原因：即使假定有完全的竞争，即使发展中国家出口制成品，也一样会处于不利的贸易条件下，而发达国家也一样能够从贸易条件中获利。  
20世纪70年代，埃及经济学家阿明在中心—外围理论和不平等交换理论的基础上，进一步论述了发展中国家的不发达问题和对发达国家的依附。阿明认为，不平等交换理论忽视了国际间的生产率差异，但伊曼纽尔提出的不平等交换情况还是存在的，因为在许多部门中，发展中国家与发达国家的工资率差异远大于劳动生产率的差异。阿明认为，建立在不平等交换基础上的国际贸易使发展中国家的经济依附于发达国家，这种依附包括贸易、金融、技术等多方面的依附。  
发展中国家在垄断资本主义的全球积累中以廉价劳动力为基础发挥着双重功能：提供廉价的出口商品，并保证发达国家的资本输出获得高利润率。发展中国家现代部门生产率的提高伴随着非现代部门的贫困化，导致经济的高度二元化。也就是说，发展中国家非现代部门的贫困大众承担着全球不平衡发展的社会代价。为了求得发展，发展中国家应该与世界资本主义体系“脱钩”，斩断与帝国主义的联系。上述几种理论尽管并不完善，并受到了不同经济学流派的批评，但毕竟对发展中国家的对外贸易政策起到了一定的指导作用，在国际谈判中也产生了广泛的影响，例如在20世纪70年代联合国贸易与发展会议上就首次倡议建立国际经济新秩序等。

# 阿明对依附论的深化
20世纪70年代，埃及经济学家阿明在中心—外围理论和不平等交换理论的基础上，进一步论述了发展中国家的不发达问题和对发达国家的依附。
阿明认为，不平等交换理论忽视了国际间的生产率差异，但伊曼纽尔提出的不平等交换情况还是存在的，因为在许多部门中，发展中国家与发达国家的工资率差异远大于劳动生产率的差异。

# 经济二元化与“脱钩”主张
发展中国家在垄断资本主义的全球积累中以廉价劳动力为基础发挥着双重功能：提供廉价的出口商品，并保证发达国家的资本输出获得高利润率。发展中国家现代部门生产率的提高伴随着非现代部门的贫困化，导致经济的高度二元化。也就是说，发展中国家非现代部门的贫困大众承担着全球不平衡发展的社会代价。为了求得发展，发展中国家应该与世界资本主义体系“脱钩”，斩断与帝国主义的联系。

# 理论的历史影响
上述几种理论尽管并不完善，并受到了不同经济学流派的批评，但毕竟对发展中国家的对外贸易政策起到了一定的指导作用，在国际谈判中也产生了广泛的影响，例如在20世纪70年代联合国贸易与发展会议上就首次倡议建立国际经济新秩序等。
